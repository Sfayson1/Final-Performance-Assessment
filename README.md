Project Description:
This program performs the following tasks:

1. Prompts the user to enter five numbers and calculates the total.
2. Prompts the user to enter five names and corresponding annual incomes.
3. Appends the new data to an existing CSV file (final.csv).
4. Creates a pie chart in Excel (final.xlsx) using the data.
5. Creates a vertical bar graph using matplotlib.

--------------------------------------------------

Files Included:

- final_exam.py → Python program file
- final.csv → Data file (provided and updated by program)
- final.xlsx → Excel file with pie chart (generated after running program)
- README.txt → Project description file

--------------------------------------------------

How to Run the Program:

1. Open IDLE.
2. Open the file: final_exam.py
3. Run the program.
4. Follow the prompts to enter numbers, names, and incomes.
5. The program will:
   - Display the total of the numbers
   - Update the CSV file
   - Generate an Excel pie chart
   - Display a bar graph

--------------------------------------------------

Requirements:

- Python 3.x
- openpyxl library
- matplotlib library

Install required libraries using:
